如何快速找到多个字典中的公共键

西班牙足球甲级联赛，每轮球员进球统计：
第一轮：{'苏亚雷斯':1, '梅西':2, '本泽马':1, 'C罗':3 ...}
第二轮：{'苏亚雷斯':2, 'C罗':1, '格里兹曼':2, '贝尔':1 ...}
第三轮：{'苏亚雷斯':1, '托雷斯':2, '贝尔':1, '内马尔':1 ...}

....

统计出前 N 轮，每场比赛都有进球的球员


Solution:

利用集合（Set）的交集操作
Step 1：使用字典的 viewkeys() 方法，得到一个字典 keys 的集合
Step 2：使用 map 函数，得到所有字典的 keys 的集合
Step 3：使用 reduce 函数，取所有字典的 keys 的集合的交集

PS
python3 中是使用 keys() 而不是 viewkeys()
